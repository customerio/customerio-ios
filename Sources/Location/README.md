# Location Module

> **⚠️ This module is under active development and is not ready for production use.**

## Status

This module is being developed as part of the Customer.io iOS SDK to provide location tracking capabilities. It is not yet published or supported.

**Do not use this module in production applications.**

## Development Milestones

- [x] Milestone 0: Module structure and configuration
- [ ] Milestone 1: Manual location API (`setLastKnownLocation`)
- [ ] Milestone 2: SDK-managed location (`requestLocationUpdateOnce`, `requestLocationUpdates`)
- [ ] Milestone 3: Location filtering and backend protection
- [ ] Milestone 4: Debugging and diagnostics
- [ ] Milestone 6: Publishing
- [ ] Milestone 7: Documentation and samples

## For Internal Testing Only

This module is included in sample apps for internal testing purposes. The CocoaPods podspec exists in the repository but is **not published** to CocoaPods trunk.
